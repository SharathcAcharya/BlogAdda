   ```bash

   # Install backend dependencies
   cd ../backend && npm install
   ```

   - Copy `.env.example` to `.env` in each folder
   - Configure your database, API keys, etc.

4. **Start development servers**:

   ```bash
   # Terminal 1: Backend

   # Terminal 2: Admin
   cd admin && npm run dev

   # Terminal 3: Frontend
   cd frontend && npm run dev
   ```
